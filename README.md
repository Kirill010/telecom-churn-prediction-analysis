# telecom-churn-prediction-analysis

# Прогнозирование оттока клиентов телеком-компании

## 📌 Описание проекта

Проект направлен на анализ и прогнозирование оттока клиентов (Churn) оператора связи. На основе исторических данных о клиентах (длительность обслуживания, тарифы, подключённые услуги, способ оплаты и т.д.) построена модель машинного обучения, которая выявляет клиентов с высоким риском ухода. Разработана стратегия удержания на 2026–2029 гг.

## 🎯 Цель

Снизить коэффициент оттока (Churn Rate) с 26,5% за счёт:
- выявления ключевых факторов ухода;
- построения модели прогнозирования;
- разработки точечных акций для клиентов с высоким риском.

## 📊 Данные

- **Объём:** 7043 записи о клиентах.
- **Признаки:** демография, услуги (интернет, ТВ, защита и др.), тип контракта, способ оплаты, ежемесячные и общие расходы, стаж (Tenure Months), Churn Score, CLTV.
- **Целевая переменная:** `Churn Label` (Yes/No).
- **Источник:** внутренние данные компании (файл `Массив данных.csv`).

## 🔧 Этапы работы

1. **Загрузка и первичный анализ**  
   - Обработка пропусков, разделителей (`,` в числах).
2. **Генерация признаков**  
   - Создание категории `Months Status` (0-12, 12-24, 24-36, 3+ years).
3. **Кодирование категориальных переменных**  
   - Использование `LabelEncoder` для перевода строк в числовой формат.
4. **Расчёт бизнес-метрик**  
   - Churn Rate, ARPU, LTV.
5. **Построение модели**  
   - Логистическая регрессия (`LogisticRegression`).
6. **Оценка качества**  
   - Accuracy, Confusion Matrix, Classification Report.
7. **Формулировка стратегии удержания** (презентация PowerPoint).

## 💡 Бизнес-выводы и стратегия удержания

### Ключевые проблемы
- 88,55% ушедших клиентов имели **помесячный контракт**.
- 45,29% ушедших оплачивали услуги через **Electronic check**.
- Клиенты без услуги **Online Security** уходят в 3 раза чаще.
- Основная причина ухода – **предложения конкурентов**.

### Предлагаемые меры (2026–2029)
1. **Акция «Переход на год»** – скидка 15% при смене помесячного контракта на годовой.
2. **Акция «Автоплатёж»** – кэшбек 5% на счёт при подключении автоплатежа с карты.
3. **Бесплатный месяц Online Security** для новых клиентов Fiber optic с автопродлением.
4. **Мониторинг риска** – еженедельная выгрузка клиентов с Churn Score > 80 для точечных предложений.

### Экономический эффект
- Снижение оттока на 5% даёт больший прирост прибыли, чем рост привлечения на 20%, за счёт увеличения LTV и сокращения маркетинговых затрат.
